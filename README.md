# Machine Learning Assignments

This repository contains a set of Machine Learning assignment notebooks.

## Assignments

### Assignment 1: Logistic Regression (Binary Classification)
- Sigmoid function and cost function
- Gradient descent optimization
- Feature mapping and regularization

### Assignment 2: Neural Networks
- Multi-layer perceptron (MLP) from scratch (NumPy)
- Forward / backward propagation
- MNIST handwritten digit classification
- Fashion-MNIST classification with PyTorch

### Assignment 3: K-Means, PCA, and CNN
- K-Means clustering (NumPy implementation)
- Principal Component Analysis (PCA)
- Convolutional Neural Networks (CNN) for Fashion-MNIST
- Dataset: `FashionMNIST_test.pth` (see the notebook for the download link)


## Setup

Install dependencies:

```bash
pip install numpy pandas matplotlib scikit-learn torch torchvision
```

Run notebooks:

```bash
jupyter notebook
```
